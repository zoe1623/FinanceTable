# FinanceTable
简介:
表格, 上下滑动固定第一行, 左右滑动固定第一列, 支持大量数据, 上下滑动时使用RecyclerView,有view的回收重用, 左右滑动使用的LinearLayout, 没有重用view, 所以横向的item不要太多, 亲测最好小于100个
